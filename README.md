⚖️ GDPR Master Quiz - 轻量级法律合规刷题系统

🚀 项目初衷
近期在准备 CIPP/E时，我发现市面上的题库系统要么过于笨重，要么不支持自定义文本导入。

作为一个法律人 × 开发者，我编写了这个单文件刷题系统。它能通过正则表达式自动解析混乱的题目文本，提供流畅的沉浸式刷题体验。

✨ 核心特性
⚡️ 零配置启动：无需数据库，下载 index.html 即可在浏览器运行。

🧠 智能解析引擎：内置 Regex 逻辑，自动识别：

复杂背景材料（SCENARIO）

题干、选项（A-D）

正确答案及深度解析（Explanation）

🌙 极客审美：支持深色模式和响应式布局。

📈 进度感知：实时计算正确率，自动保存进度到 localStorage，刷新不丢失。

🏗️ 场景化支持：特别针对法律考试中的“一案多题（Scenario-based questions）”进行了 UI 优化。

🛠️ 技术实现细节
本项目体现了法律逻辑向代码逻辑的转化：

数据层：使用 localStorage API 实现前端数据持久化。

解析层：采用状态机思想编写的 parseText 函数，能够兼容多种换行符和混排格式。

UI 层：基于 CSS Variables 实现动态主题切换，利用 CSS Animations 提升交互平滑度。

📖 如何使用
复制粘贴：将你的题库文本（甚至是从 PDF 复制出来的）直接贴入“设置”区域。

开始刷题：系统会自动将文本切割为题目对象。

复习解析：回答后立即显示解析，支持快速跳转。
